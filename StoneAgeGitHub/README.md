# StoneAge Minecraft Server Plugin

Plugin หลักสำหรับเซิร์ฟเวอร์ Minecraft ยุคหิน 1.20.x

## ระบบที่มี
- ☠️ ระบบกล่องศพ (ทุบได้ของ ไม่ได้กล่อง)
- 🪓 โต๊ะคราฟมีโอกาสสำเร็จ/ล้มเหลว
- 💰 ระบบเงินและขายของ
- 🎮 เชื่อม Discord (ชื่อในเกม = ชื่อ Discord)
- 🔒 ระบบ Whitelist ตามยศ Discord
